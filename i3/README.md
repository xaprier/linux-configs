# i3 Settings

## Dependencies
- i3
- polybar
- rofi(catppuccin)
- playerctl
- picom
- feh(wallpaper-change)
- lm_sensors
- btop
- pamixer
- nitrogen(wallpaper-slide)
- nerd-fonts-git

```sh
yay -S i3-wm polybar rofi playerctl picom feh lm_sensors btop pamixer nitrogen nerd-fonts-git
```
